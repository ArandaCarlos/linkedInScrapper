## Creación de un entorno virtual (venv)

Para crear un entorno virtual (venv) en Python, seguir los siguientes pasos:

    Abrir la consola o terminal en el sistema operativo.
    Navegar hasta la carpeta en la que se desea crear el entorno virtual.
    Ejecutar el comando python -m venv nombre_del_entorno para crear el entorno virtual. Reemplazar "nombre_del_entorno" con el nombre deseado para el entorno.

## Activación del entorno virtual

Para activar el entorno virtual recién creado, seguir los siguientes pasos:

    Abrir la consola o terminal en el sistema operativo.
    Navegar hasta la carpeta en la que se encuentra el entorno virtual.
    Ejecutar el comando source nombre_del_entorno/bin/activate para activar el entorno virtual. Reemplazar "nombre_del_entorno" con el nombre del entorno creado anteriormente.

## Ejecución del archivo .py

Una vez activado el entorno virtual, se puede ejecutar el archivo .py con el siguiente comando:

    python nombre_del_archivo.py

Reemplazar "nombre_del_archivo" con el nombre del archivo .py que se desea ejecutar.
Desactivación del entorno virtual

Para desactivar el entorno virtual, ejecutar el comando deactivate.

Es importante recordar que cada vez que se abra una nueva sesión en la consola o terminal, el entorno virtual deberá ser activado nuevamente antes de ejecutar el archivo .py.
